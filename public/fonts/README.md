# Aeonik Font Files

## Required Files

Place the following Aeonik font files in this directory:

### Regular Weight (400)
- `Aeonik-Regular.woff2`
- `Aeonik-Regular.woff`

### Medium Weight (500)
- `Aeonik-Medium.woff2`
- `Aeonik-Medium.woff`

### Bold Weight (700)
- `Aeonik-Bold.woff2`
- `Aeonik-Bold.woff`

## Where to Get Aeonik

Aeonik is a premium font. You can:
1. Purchase from the official Aeonik website
2. Use your existing Aeonik license
3. Contact your design team for the font files

## Installation

Once you have the font files:
1. Copy all `.woff2` and `.woff` files to this directory
2. The fonts will automatically load via `/src/fonts.css`
3. Refresh your browser to see the changes

## Fallback

If you don't have Aeonik fonts, the site will fall back to:
- `system-ui` (macOS: San Francisco, Windows: Segoe UI)
- `-apple-system`
- `sans-serif`

This ensures the site still looks good while you obtain the Aeonik font files.
